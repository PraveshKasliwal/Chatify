# Chatify

A real-time chat application built using **React, Node.js, Express, MongoDB, and Socket.IO**. Chatify enables users to communicate instantly through private and group chats with persistent message storage and AI-powered conversation summaries.

---

## 🚀 Live Demo
🔗 https://chatify-frontend-kappa.vercel.app/

---

## ✨ Features
- 💬 Real-time messaging using **Socket.IO**
- 👥 **Private and Group Chats**
- 🔐 **User Authentication**
- 🗂 **Message persistence in MongoDB**
- 🤖 **AI Summarization:** Select messages and generate a short summary of the conversation using **Google Gemini API**
- ⚡ Fast frontend built with **React + Vite**

## 📦 Installation

### 1️⃣ Clone the Repository
```bash
git clone https://github.com/PraveshKasliwal/Chatify.git
cd Chatify
```

### 2️⃣ Install Dependencies
```bash
npm install
```

---

## ▶️ Running the Application

### Start Backend
```bash
cd backend
npm install
npm run dev
```

### Start Frontend
```bash
cd frontend
npm install
npm start
```
