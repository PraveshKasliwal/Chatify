# Chatify

A real-time chat app built with React, Node.js, Express, MongoDB, and Socket.IO.

## Features
- Real-time messaging
- Group and private chats
- User authentication
- Message persistence in MongoDB
- - **AI Summarization:** Select messages and generate a short summary of the conversation
